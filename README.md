# Bouncing Balls

Whenever you click in the window, a circle is fired from the clicked position at a random speed and angle. It also handles multiple balls being fired and bouncing at the same time.

**Tech Stack:**

* ES6
* Jasmine
* Karma
* Webpack

## Getting Started

**Installing:**

`yarn`

\
**Running app:**

`yarn start`

\
**Running test:**

`yarn test`
