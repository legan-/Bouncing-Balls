# Bouncing Balls

Whenever you click in the window, a ball is fired from the clicked position at a random speed and angle. It also handles multiple balls being fired and bouncing at the same time.

**Tech Stack:**

- ES6
- Jest
- Webpack

## Getting Started

**Installation:**

`yarn`

\
**Running the app:**

`yarn start`

\
**Running tests:**

`yarn test`
